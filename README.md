## 👋 Introduction

**Splitty** is a simple tool that helps you organize and split massive folders that are too big to fit into one place. If you have a huge directory filled with thousands of loose files and want to burn them onto multiple DVDs, Blu-rays, or pack them into external drives, this app automatically calculates the space and splits everything into folders of the exact size you need.

## 💡 Motivation

Distributing hundreds of gigabytes of loose files or deeply nested directory structures across physical media targets requires precise mathematical calculation to maximize storage efficiency. Manually selecting, grouping, and calculating the total size of specific folder branches to hit exact media limits without leaving wasted space or overfilling a volume is an incredibly tedious bottleneck.

Instead of trial-and-error manual sorting or writing volatile batch scripts, this application automatically maps the entire input directory and computes the optimal file distribution layout based on predefined optical standards or custom sector constraints.

##### ⚡ The Real Question
###### How much time do you lose trying to manually balance and split a 100 GB nested directory structure into exact storage blocks without leaving dead space or exceeding target capacity limits?.

> [!NOTE]
> This is an unmaintained legacy project. The source-code relies on older development standards and will not be refactored, but the utility remains fully functional and reliable for its intended purpose.

## 🤖 Features

- **Automatic Splitting:** Analyzes your heavy folder and splits it into multiple blocks that fit perfectly into the destination.
- **Ready-To-Use Profiles:** Includes standard sizes for CDs, DVDs, and Blu-rays so you don't have to look up exact media capacities.
- **Custom Sizes:** You can set any megabyte or gigabyte limit you want if you are targeting USB flash drives or external HDDs.
- **No Forced Compression:** It copies your files exactly as they are, keeping your original folder structure intact.
- **Output Format Layouts:** Optionally, it can organize the prepared blocks ready for these formats (without compressing them):
  - 7Z
  - RAR
  - SFX (WinRAR Self-Extractor)
  - ISO
  - TAR
